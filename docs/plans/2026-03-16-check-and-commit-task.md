# 任务清单 - 代码检测与提交

- [ ] 工作区状态分析 [/]
    - [ ] 运行 `git status` 获取完整变更列表
    - [ ] 检查是否存在敏感信息或不应提交的文件 (如 .env, tmp 文件)
- [ ] 代码质量检测 [ ]
    - [ ] 后端编译与单元测试 (nexusarchive-java)
    - [ ] 前端编译 (如有变更)
    - [ ] 检查是否符合信创合规要求 (SM3, BigDecimal 等)
- [ ] 文档一致性检查 [ ]
    - [ ] 检查 `README.md` 是否已按代码变更更新
    - [ ] 更新 `docs/CHANGELOG.md`
- [ ] 提交与推送 [ ]
    - [ ] `git add` 所有变更
    - [ ] 编写符合规范的 Commit Message
    - [ ] 执行 `git push`
