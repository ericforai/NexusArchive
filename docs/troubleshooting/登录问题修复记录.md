# 登录问题修复记录

> **日期**: 2025-12-07  
> **问题**: 用户无法登录，登录后立即被踢回登录页  
> **状态**: ✅ 已解决

---

## 问题现象

1. 用户输入正确的用户名密码（admin / admin123）
2. 后端 `/auth/login` 返回 **200 OK**
3. 前端显示 "Login success, navigating to: /system"
4. 但页面**没有跳转**，仍停留在登录页
5. 后续 API 调用（如 `/api/stats/dashboard`）全部返回 **401 Unauthorized**

---

## 根因分析

### 存储机制不一致

项目中存在**两套 Token 存储机制**，导致登录状态无法正确传递：

| 组件 | 使用的存储方式 | 存储 Key |
|------|---------------|---------|
| `LoginView` | `useAuthStore` (Zustand) | `nexus-auth` (JSON 格式) |
| `ProtectedRoute` | `safeStorage` (直接 localStorage) | `token` (字符串) |
| `client.ts` | `useAuthStore.getState().token` | `nexus-auth` |

### 问题链路

```
登录成功 → Token 存入 Zustand (nexus-auth)
         → ProtectedRoute 检查 safeStorage.getItem('token')
         → 返回 null（因为 key 不匹配）
         → 判定为未登录
         → 重定向到 /system/login
```

---

## 解决方案

### 修改 ProtectedRoute.tsx

**Before** (错误做法):
```tsx
import { safeStorage } from '../../utils/storage';

const checkAuth = async () => {
    const token = safeStorage.getItem('token');  // ❌ 直接读取 localStorage
    const cachedUser = safeStorage.getItem('user');
    // ...
};
```

**After** (正确做法):
```tsx
import { useAuthStore } from '../../store/useAuthStore';

const { token, user, isAuthenticated, login, logout } = useAuthStore();

const checkAuth = async () => {
    if (token && user) {  // ✅ 从 Zustand store 读取
        console.log('[ProtectedRoute] Token and user found in store');
        setIsChecking(false);
        return;
    }
    // ...
};
```

### 核心原则

> **单一数据源 (Single Source of Truth)**  
> 所有认证状态的读写必须通过同一个 Store（`useAuthStore`），不能混用多种存储方式。

## 经验教训

### 1. 状态管理一致性

当项目从手动管理状态（如 `safeStorage`）迁移到状态管理库（如 Zustand）时：

- ✅ **必须全面替换**，不能部分组件用新方式、部分用旧方式
- ✅ 建议使用 **全局搜索** 找出所有 `safeStorage.getItem('token')` 的调用点
- ✅ 迁移后应**删除或标记废弃**旧的存储工具

### 2. 调试技巧

当登录 API 返回 200 但仍无法进入系统时，排查顺序：

1. **检查 Token 存储位置**：登录后 localStorage 中的实际 key 是什么？
2. **检查路由守卫**：它从哪里读取 Token？key 是否匹配？
3. **检查 Axios 拦截器**：请求头是否正确携带 Token？

### 3. Zustand persist 的 key 格式

使用 Zustand `persist` 中间件时，数据存储格式为：

```json
{
  "state": {
    "token": "xxx",
    "user": {...},
    "isAuthenticated": true
  },
  "version": 0
}
```

**不能**直接用 `localStorage.getItem('token')` 读取！必须通过 `useAuthStore.getState().token`。
