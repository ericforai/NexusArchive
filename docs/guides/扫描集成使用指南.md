# 扫描集成使用指南

## 功能概述

扫描集成模块提供完整的纸质凭证数字化解决方案，支持多种输入方式：

| 功能 | 说明 |
|------|------|
| **文件上传** | 拖拽或点击上传本地文件 |
| **OCR 识别** | 自动识别发票、合同、银行回单等单据 |
| **移动端扫码** | 手机扫码上传照片 |
| **监控文件夹** | 扫描仪保存文件后自动导入 |

---

## 1. 文件上传

### 访问路径

```
http://localhost:15175/system/collection/scan
```

或从菜单进入：**资料收集** → **扫描集成**

### 上传方式

**方式一：拖拽上传**
- 直接将文件拖入上传区域

**方式二：点击上传**
- 点击上传区域，选择本地文件

**支持的文件类型**
- PDF 文档
- 图片（JPG、JPEG、PNG）
- 其他扫描格式

### 上传后处理

文件上传后自动进入工作区，系统会：
1. 显示文件列表
2. 自动触发 OCR 识别
3. 提取关键信息（发票号、金额、日期等）
4. 支持人工编辑修正

---

## 2. OCR 智能识别

### 识别的单据类型

| 类型 | 说明 |
|------|------|
| **增值税发票** | 自动识别发票代码、号码、金额、税额 |
| **合同协议** | 识别合同主体、金额、日期 |
| **银行回单** | 识别交易流水、金额、日期 |
| **身份/资质证件** | 识别证件信息 |
| **其他** | 通用文本识别 |

### OCR 操作流程

```
1. 上传文件
   ↓
2. 自动 OCR 识别（等待几秒）
   ↓
3. 查看识别结果
   ↓
4. 人工编辑/修正（如有需要）
   ↓
5. 提交到预归档池
```

### 人工编辑

识别结果可以手动修改：
- 点击任意字段进入编辑模式
- 修改后自动保存
- 支持添加自定义字段

## 3. 移动端扫码上传

### 功能说明

当扫描仪不支持或使用手机拍照时，可以通过移动端上传：

1. **在电脑上生成二维码**
2. **用手机扫描二维码**
3. **手机上传照片**
4. **电脑端自动同步**

### 使用步骤

#### 步骤 1：创建上传会话

在扫描集成页面，点击 **「手机扫码」** 按钮：

```
┌─────────────────────────────────┐
│   📱 手机扫码上传                │
│                                 │
│   [生成二维码]                   │
│                                 │
│   ┌─────────┐                   │
│   │  ████   │                   │
│   │  █  █   │                   │
│   │  ████   │                   │
│   │         │                   │
│   └─────────┘                   │
│                                 │
│   会话 ID: abc123...            │
│   有效期: 30 分钟                │
│                                 │
│   [关闭]                         │
└─────────────────────────────────┘
```

#### 步骤 2：手机扫码上传

1. 用手机扫描二维码
2. 打开上传页面
3. 拍照或选择相册照片
4. 点击上传

#### 步骤 3：电脑端同步

上传成功后，文件会自动出现在电脑端的工作区列表中。

### 注意事项

- 会话有效期 **30 分钟**
- 每次生成二维码会创建新的会话
- 确保手机和电脑在同一网络

## 4. 监控文件夹

### 功能说明

配置扫描仪软件，将扫描文件保存到指定文件夹，系统会自动检测并导入。

### 配置步骤

#### 步骤 1：打开监控设置

在扫描集成页面，点击 **「监控文件夹」** 按钮。

#### 步骤 2：添加监控

```
┌─────────────────────────────────────┐
│ 📁 监控文件夹设置                     │
│                                     │
│ 添加新监控                           │
│                                     │
│ 文件夹路径 *                         │
│ [/home/user/Documents/Scan]        │
│                                     │
│ 这是服务器上的文件系统路径，           │
│ 请确保后端服务有访问权限              │
│                                     │
│ 文件类型过滤                         │
│ [*.pdf;*.jpg;*.jpeg;*.png]         │
│                                     │
│ 使用分号分隔多个文件类型，支持通配符 * │
│                                     │
│ ☐ 导入后删除源文件                   │
│                                     │
│ [添加监控]                           │
└─────────────────────────────────────┘
```

#### 步骤 3：配置扫描仪

在扫描仪软件中设置：
1. **保存格式**：PDF 或 JPG
2. **保存路径**：填入配置的文件夹路径
3. **扫描设置**：分辨率 300DPI，彩色/黑白

#### 步骤 4：自动导入

扫描仪保存文件后，系统会：
1. 自动检测新文件
2. 等待文件写入完成
3. 检查文件类型过滤
4. 导入到工作区
5. 触发 OCR 识别

### 监控管理

| 操作 | 说明 |
|------|------|
| **启用/暂停** | 点击圆圈图标切换监控状态 |
| **编辑** | 修改过滤规则或路径 |
| **删除** | 移除不需要的监控 |

### 文件类型过滤示例

```
*.pdf                    # 仅 PDF
*.pdf;*.jpg              # PDF 和 JPG
*.pdf;*.jpg;*.png        # PDF、JPG、PNG
*.jpeg;*.png             # JPEG 和 PNG
```

### 服务器路径示例

| 系统 | 路径示例 |
|------|----------|
| **Linux** | `/home/user/Documents/Scan` |
| **Windows** | `C:\Users\user\Documents\Scan` |
| **网络存储** | `/mnt/scanner/files` |

## 5. 提交到预归档池

### 操作流程

```
工作区 → 识别/编辑 → 提交 → 预归档池
```

### 提交条件

- OCR 识别已完成
- 必填字段已填写
- 文件完整性校验通过

### 提交后

文件进入预归档池，可以：
- 继续编辑元数据
- 与记账凭证关联
- 执行正式归档

## 6. 常见问题

### Q1: 上传后没有触发 OCR？

**A:** 请检查：
- 后端 OCR 服务是否正常运行
- 文件格式是否支持
- 文件是否损坏

### Q2: 手机扫码后无法上传？

**A:** 请检查：
- 二维码是否过期（30 分钟有效期）
- 手机和电脑是否在同一网络
- 浏览器是否阻止了摄像头权限

### Q3: 监控文件夹没有自动导入？

**A:** 请检查：
- 文件夹路径是否正确
- 后端服务是否有该路径的读取权限
- 文件类型过滤是否包含该文件类型
- 监控是否处于启用状态

### Q4: OCR 识别结果不准确？

**A:** 
- 支持人工编辑修正
- 可以使用更高分辨率的扫描（300 DPI 以上）
- 确保扫描图像清晰、无倾斜

## 7. API 接口

### 工作区管理

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/scan/workspace` | GET | 获取工作区文件列表 |
| `/api/scan/workspace/upload` | POST | 上传文件到工作区 |
| `/api/scan/workspace/{id}/ocr` | POST | 触发 OCR 识别 |
| `/api/scan/workspace/{id}` | PUT | 更新工作区数据 |
| `/api/scan/workspace/{id}/submit` | POST | 提交到预归档池 |
| `/api/scan/workspace/{id}` | DELETE | 删除工作区项 |

### 移动端会话

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/scan/mobile/session` | POST | 创建移动端上传会话 |
