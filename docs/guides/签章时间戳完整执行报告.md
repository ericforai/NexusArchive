## 🔧 配置要求

### 必需配置

1. **密钥库文件**
   - 路径: `./keystore/signature.p12`（或配置的路径）
   - 格式: PKCS12
   - 生成方法: 使用 `generate_keystore.sh` 脚本

2. **application.yml 配置**
   ```yaml
   signature:
     keystore:
       path: ./keystore/signature.p12
       password: changeit
   
   timestamp:
     enabled: false  # 或 true（如果配置了TSA）
     fallback-on-error: true
   ```

### 可选配置

1. **TSA 服务**（如果需要真实时间戳）
   ```yaml
   timestamp:
     enabled: true
     tsa:
       url: http://tsa.example.com/tsp
   ```

2. **自动执行**（归档时自动签章和时间戳）
   ```yaml
   signature:
     auto-sign: true
   
   timestamp:
     auto-request: true
   ```
