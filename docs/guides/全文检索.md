# 全文检索功能

## 概述

NexusArchive 集成 Elasticsearch 提供高性能全文检索，支持中文分词和搜索结果高亮。

---

## 功能特性

| 功能 | 说明 |
|:---|:---|
| 中文分词 | 使用 IK 分词器 |
| 高亮显示 | 搜索关键词高亮 |
| 多字段搜索 | 标题、档号、OCR 文本 |
| 异步索引 | 不阻塞主业务流程 |

---

## 配置

### application.yml

```yaml
spring:
  elasticsearch:
    uris: ${ELASTICSEARCH_URIS:http://localhost:9200}
    username: ${ELASTICSEARCH_USERNAME:}
    password: ${ELASTICSEARCH_PASSWORD:}
```

### 索引结构

```json
{
  "mappings": {
    "properties": {
      "archiveCode": { "type": "keyword" },
      "title": { 
        "type": "text",
        "analyzer": "ik_max_word",
        "search_analyzer": "ik_smart"
      },
      "ocrText": { "type": "text", "analyzer": "ik_max_word" },
      "fiscalYear": { "type": "integer" },
      "retentionPeriod": { "type": "keyword" }
    }
  }
}
```

---

## API 接口

### 全文搜索

```http
GET /api/search?keyword=发票&page=0&size=20
Authorization: Bearer {token}
```

**响应**:
```json
{
  "code": 200,
  "data": {
    "total": 42,
    "list": [
      {
        "id": "xxx",
        "document": { "title": "增值税发票", ... },
        "highlights": { "title": ["增值税<em>发票</em>"] },
        "score": 5.8
      }
    ]
  }
}
```

### 重建索引

```http
POST /api/search/reindex
Authorization: Bearer {token}
```

> 需要 ADMIN 角色权限

## 部署要求

| 组件 | 版本要求 |
|:---|:---|
| Elasticsearch | 8.x |
| IK 分词器 | 与 ES 版本匹配 |

### IK 分词器安装

```bash
# 进入 ES 容器或安装目录
bin/elasticsearch-plugin install \
  https://github.com/medcl/elasticsearch-analysis-ik/releases/download/v8.11.0/elasticsearch-analysis-ik-8.11.0.zip
```

---

## 注意事项

1. **可选组件** - ES 不可用时系统仍可正常运行
2. **索引延迟** - 新建档案后 1-2 秒内完成索引
3. **存储空间** - 预留约为数据库 1.5 倍的磁盘空间
