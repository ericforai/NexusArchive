# 电子会计档案系统测试用例表（含自动化/压测脚本清单）

## 1. 范围与优先级
- P0：档案归档与存储、四性检测、电子签章/时间戳、权限与审计、检索与调阅、备份与恢复、ERP/财务对接、安全（SQL 注入/XSS/鉴权/越权）。
- P1：批量导入/导出、档案借阅/销毁流程、OCR/元数据、报表统计、版本管理。
- P2：界面体验、消息通知、个人偏好（基础覆盖）。

## 2. 核心功能用例表（优先覆盖）

| AR-02 | 归档 | P0 | 存储空间耗尽保护 | 设置磁盘 95%+，继续归档 | 返回失败并提示；无脏数据；无静默 | 接口+模拟磁盘 |

| SR-02 | 检索 | P0 | 索引一致性 | 删除/更新后立即检索 | 不返回幽灵记录；索引刷新 | 接口自动化 |

## 5. 自动化脚本清单（建议命名与触发）
- `api/archive_concurrency_spec`: 并发归档、防覆盖（P0，CI 每次）。
- `api/storage_quota_guard_spec`: 磁盘阈值、特殊字符归档（P0，每日）。
- `api/authz_audit_spec`: 越权、审计不可删、失败日志（P0，CI）。
- `api/four_integrities_spec`: 哈希一致/TSA 超时/结果持久化（P0，CI）。
- `api/sign_timestamp_spec`: 证书过期、签章坐标、时区校验（P0，CI）。
- `api/search_index_spec`: 检索性能基线、索引一致性（P0，夜间）。
- `api/backup_restore_spec`: 备份校验、增量链、权限/索引恢复（P0，周）。
- `api/erp_sync_spec`: 超时重试、幂等、防重复、字段映射（P0，CI）。
- `api/batch_import_export_spec`: 分批导入/导出、模板校验、失败明细（P1，夜间）。
- `api/workflow_borrow_destroy_spec`: 借阅/销毁审批流、日志校验（P1，夜间）。
- `api/ocr_metadata_spec`: 置信度阈值、回退人工校正（P1，周）。
- `api/report_version_spec`: 报表口径、版本冲突/不可改（P1，周）。
- 安全扫描：`security/owasp_suite`（SQLi/XSS/CSRF/IDOR，P0，周）；`security/password_policy_spec`（暴破/弱口令，P0，周）。

## 6. 压测脚本清单（示例）
- `perf/search_peak.jmx`：50 并发检索，持续 2h，TP95<2s。
- `perf/archive_soak.jmx`：72h 持续归档，观察内存/FD/线程泄漏。
- `perf/import_100k.jmx`：单次导入 10 万，记录成功率/RT/CPU/IO。
- `perf/upload_1gb.jmx`：1GB 单文件上传/预览延迟，带断点/重试。
- `perf/erp_flaky_network.jmx`：高延迟/抖动/断连，对接重试与幂等。
- `perf/backup_restore_cycle.sh`：全量+增量备份恢复链路时延与成功率。
- `perf/upgrade_rollback.sh`：升级→回滚全过程时长、数据一致性校验。

## 7. 数据与环境准备
- 文件集：10KB/10MB/500MB/1GB，PDF/Excel/扫描件，中文/英文/特殊字符/超长文件名。
- 账号矩阵：普通/财务/审计/管理员；过期/禁用账号；多组织隔离。
- 对接模拟：ERP 沙箱+延迟/抖动注入；过期/吊销证书；TSA 开关。
- 存储：设置磁盘阈值 80%/95%；多节点/多副本；索引节点独立监控。

## 8. 验收/退出准则
- P0 全部通过；安全/合规高危=0；归档成功率≥99.9%，检索 TP95<2s。
- 备份恢复、升级/回滚演练通过；ERP 丢单=0；审计日志 100% 覆盖且不可篡改。
- P1 核心通过，性能基线达标；P2 基础体验无阻塞缺陷。

## 9. 脚本骨架示例（Playwright + k6）
> 需安装：`npm i -D @playwright/test`；k6 请使用本地/CI 安装二进制。示例使用环境变量（`BASE_URL`、`TOKEN` 等）。

### Playwright API 测试骨架
`tests/playwright/api/archive_concurrency.spec.ts`
```ts
import { test, expect, request } from '@playwright/test';

const BASE_URL = process.env.BASE_URL ?? 'http://localhost:3000';
const TOKEN = process.env.TOKEN ?? 'dev-token';

test.describe('归档并发与存储保护', () => {
  test('同档案并发上传仅保留一版且有告警', async () => {
    const ctx = await request.newContext({
      baseURL: BASE_URL,
      extraHTTPHeaders: { Authorization: `Bearer ${TOKEN}` },
    });
    const payload = { voucherNo: 'INV-001', filename: 'invoice.pdf', content: '...base64...' };
    const uploads = await Promise.allSettled(
      Array.from({ length: 5 }, () => ctx.post('/api/archive', { data: payload })),
    );
    const successes = uploads.filter(
      r => r.status === 'fulfilled' && (r as PromiseFulfilledResult<any>).value.ok(),
    ).length;
    expect(successes).toBe(1);
    const audit = await ctx.get('/api/audit/logs?voucherNo=INV-001');
    expect(audit.ok()).toBeTruthy();
  });

  test('存储耗尽时阻断写入并告警', async ({ request }) => {
    await request.post('/api/admin/storage/simulate', { data: { usage: 0.97 } });
    const res = await request.post('/api/archive', { data: { voucherNo: 'INV-002' } });
    expect(res.status()).toBe(507);
    expect(await res.json()).toMatchObject({ code: 'STORAGE_FULL' });
  });
});
```

`tests/playwright/api/four_integrities.spec.ts`
```ts
import { test, expect } from '@playwright/test';

test('哈希算法一致性与 TSA 超时降级', async ({ request }) => {
  const upload = await request.post('/api/archive', { data: { hashAlg: 'MD5', file: '...' } });
  expect(upload.ok()).toBeTruthy();

  await request.post('/api/admin/tsa/toggle', { data: { enabled: false, delayMs: 8000 } });
  const verify = await request.post('/api/archive/verify', { data: { hashAlg: 'SHA256', id: 'id-123' } });
  expect(verify.status()).toBe(400);
  expect(await verify.json()).toMatchObject({ code: 'HASH_MISMATCH' });

  const audit = await request.get('/api/audit/logs?target=id-123');
  expect(audit.ok()).toBeTruthy();
});
```

### k6 压测脚本骨架
`perf/search_peak.k6.js`
```js
import http from 'k6/http';
import { check, sleep } from 'k6';

export const options = {
  vus: 50,
  duration: '2h',
  thresholds: { http_req_duration: ['p(95)<2000'] },
};

const BASE_URL = __ENV.BASE_URL || 'http://localhost:3000';

export default function () {
  const res = http.get(`${BASE_URL}/api/search?q=发票`);
  check(res, { 'status 200': r => r.status === 200 });
  sleep(1);
}
```

`perf/archive_soak.k6.js`

import http from 'k6/http';

`perf/upload_1gb.k6.js`
```js
import http from 'k6/http';
import { check } from 'k6';
import { SharedArray } from 'k6/data';

const BASE_URL = __ENV.BASE_URL || 'http://localhost:3000';
const file = new SharedArray('file', () => [open('./fixtures/1gb.bin', 'b')]);

export const options = {
  vus: 1,
  iterations: 5,
  thresholds: { http_req_duration: ['p(95)<15000'] },
};

export default function () {
  const res = http.post(
    `${BASE_URL}/api/archive/upload`,
    { file: http.file(file[0], 'bigfile.bin', 'application/octet-stream') },
  );
  check(res, { 'status 200': r => r.status === 200 });
}
```
