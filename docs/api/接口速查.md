# API 接口速查表

本文档提供 NexusArchive 系统的常用 API 接口速查。

---

## 认证接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/auth/login` | POST | 用户登录 |
| `/api/auth/me` | GET | 获取当前用户信息 |
| `/api/auth/logout` | POST | 用户登出 |
| `/api/auth/refresh` | POST | 刷新 Token |

---

## 统计接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/stats/dashboard` | GET | 仪表盘统计数据 |
| `/api/stats/storage` | GET | 存储占用统计 |
| `/api/stats/archival-trend` | GET | 归档趋势（近14日） |
| `/api/stats/ingest-status` | GET | SIP 任务状态分布 |

---

## 档案关联接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/relations/{id}/graph` | GET | 获取档案关系图谱 |
| `/api/relations/{id}/files` | GET | 获取已关联文件 |
| `/api/relations/{id}/compliance` | GET | 获取四性合规状态 |

---

## 库房控制接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/warehouse/shelves` | GET | 获取架位列表 |
| `/api/warehouse/environment` | GET | 获取环境数据 |
| `/api/warehouse/racks/{id}/command` | POST | 发送架位命令 |

命令 body 格式：
```json
{ "action": "OPEN|CLOSE|VENT|LOCK|UNLOCK" }
```

---

## 符合性检查接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/compliance/archives/{id}` | GET | 获取单个档案符合性状态 |
| `/api/compliance/archives/batch` | POST | 批量检查符合性 |
| `/api/compliance/archives/{id}/report` | GET | 获取符合性检查报告 |
| `/api/compliance/statistics` | GET | 获取符合性统计数据 |

---

## 借阅接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/borrowing` | POST | 创建借阅申请 |
| `/api/borrowing` | GET | 分页查询借阅列表 |
| `/api/borrowing/{id}/approve` | POST | 审批通过/拒绝 |
| `/api/borrowing/{id}/return` | POST | 归还档案 |
| `/api/borrowing/{id}/cancel` | POST | 取消申请 |

创建借阅 body:
```json
{
  "archiveId": "档案ID",
  "reason": "借阅原因（可选）",
  "expectedReturnDate": "预计归还日期（可选）"
}
```

审批 body:
```json
{
  "approved": true/false,
  "comment": "审批意见（可选）"
}
```

## 用户管理接口

需 `manage_users` 权限或 `system_admin` 角色：

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/admin/users` | GET | 分页查询用户 |
| `/api/admin/users` | POST | 创建用户 |
| `/api/admin/users/{id}` | PUT | 更新用户 |
| `/api/admin/users/{id}` | DELETE | 删除用户 |
| `/api/admin/users/{id}/reset-password` | POST | 重置密码 |
| `/api/admin/users/{id}/status` | PUT | 切换状态 |
