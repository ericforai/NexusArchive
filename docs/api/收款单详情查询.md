收款单详情查询
发布时间:2025-07-03 22:03:41
支持收款单根据单据主键ID查询详情
1. 请求说明
请求域名	开放API: 动态域名，获取方式详见 获取租户所在数据中心域名
集成API: 详细域名信息，请见 连接配置
请求地址	https://dbox.yonyoucloud.com/iuap-api-gateway/yonbip/EFI/collection/detail
请求方式	GET
ContentType	application/json
应用场景	开放API/集成API
API类别	详情查询
事务和幂等性	无
2. 请求参数
只看必填项
名称	类型	参数位置	必填	描述
access_token	string	query	是	调用方应用token
企业自建获取token
服务商获取token
id	string	query	是	收款单id    示例: 2326431654121728
3. 请求示例
Url: /yonbip/EFI/collection/detail?access_token=访问令牌&id=2326431654121728
