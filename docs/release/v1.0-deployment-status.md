```bash
# 开发环境已运行，可以直接使用

# 后端: http://localhost:19090/api
```

### 方案3: 手动构建镜像

```bash
# 构建后端镜像
cd nexusarchive-java
docker build -t nexusarchive-backend:v1.0.0 .

# 构建前端镜像
cd ..
docker build -f Dockerfile.frontend.prod -t nexusarchive-web:v1.0.0 .

# 然后启动服务
docker-compose -f docker-compose.prod.yml --env-file .env.prod up -d
```

### 方案4: 使用代理或镜像源

如果网络环境受限，可以配置Docker镜像源：

```bash
# 配置Docker镜像加速器（中国镜像）
# 编辑 /etc/docker/daemon.json
{
  "registry-mirrors": [
    "https://docker.mirrors.ustc.edu.cn",
    "https://hub-mirror.c.163.com"
  ]
}

# 重启Docker服务
sudo systemctl restart docker
```
