# 电子会计档案系统 v1.0 发布说明

> **版本**: v1.0  
> **发布日期**: 2025-01  
> **发布类型**: 正式发布

---

## 📋 发布概览

电子会计档案系统 v1.0 是系统的第一个正式版本，完成了核心电子档案管理流程与合规能力，包括全宗隔离、审计链路、四性检测、水印预览等核心功能。

---

## ✨ 核心功能

### 1. 档案归集与管理
- ✅ 电子归档与关联
- ✅ 结构化元数据存储
- ✅ 全宗隔离机制
- ✅ 核心元数据模型

### 2. 检索与利用
- ✅ 高级检索与脱敏
- ✅ 流式预览与动态水印
- ✅ JSONB索引优化

### 3. 四性检测与合规
- ✅ 真实性检测（文件哈希 + 数字签名）
- ✅ 完整性检测（元数据校验）
- ✅ 可用性检测（Magic Number + Tika）
- ✅ 安全性检测（病毒扫描）

### 4. 审计与证据
- ✅ 审计哈希链（SM3）
- ✅ 审计日志验真
- ✅ 证据包导出
- ✅ 防伪水印（服务端流式渲染）

### 5. 借阅与流程
- ✅ 借阅状态机（完整流转）
- ✅ 跨全宗授权票据
- ✅ 审批流程

### 6. 数据迁移
- ✅ CSV/Excel导入
- ✅ 字段映射配置
- ✅ 数据验证与错误报告
- ✅ 事务回滚支持

## 🎯 技术特性

### 后端技术栈
- Spring Boot 3.x
- MyBatis Plus
- PostgreSQL（当前仅支持PostgreSQL）
- BouncyCastle（国密SM2/SM3）
- Apache Tika（文件类型检测）
- PDFBox（PDF处理）
- OFD Parser（OFD文件解析）

### 前端技术栈
- React 18+
- TypeScript
- Tailwind CSS
- Ant Design
- React Router

### 安全特性
- 全宗隔离（MyBatis拦截器强制过滤）
- 三员分立（SysAdmin/SecAdmin/AuditAdmin）
- 审计哈希链（不可篡改）
- 动态水印（防伪与追责）
- 数据脱敏（敏感字段保护）

## 📝 已知问题与限制

### 已知限制
1. **数据库支持**: 当前仅支持PostgreSQL，达梦/金仓数据库适配层已暂停开发
2. **实物档案管理**: 不包含在v1.0交付范围（后续版本规划）
3. **销毁流程增强**: 部分功能为P2项，不纳入v1.0

### 已知问题
- 无严重已知问题
- 部分测试用例因环境问题可能偶发超时（非功能问题）
