版本: 2.2 适用对象: 档案管理员、财务人员、系统管理员 最后更新: 2025-12-04

---
1. 系统简介
数凭电子会计档案系统 (EAA) 是一款专为企业级财务归档设计的私有化部署解决方案。系统严格遵循国家档案局 DA/T 94-2022《电子会计档案管理规范》，深度适配信创环境，提供从凭证采集、四性检测、归档整理到长期保存的全链路管理能力。

3. 核心业务流程 (End-to-End Workflow)
本章节将引导您完成从"凭证接收"到"正式归档"的完整业务闭环。
步骤 1: 凭证采集 (Ingest)
所有的电子会计资料首先进入预归档库。
1. 进入 [预归档] -> [电子凭证池]。
2. 点击 "上传"，选择您的 OFD/PDF 发票或 XML 凭证文件。
3. 系统会自动进行： 哈希计算: 使用 SM3 算法（信创合规）计算文件指纹 OCR 识别: 自动识别文件内容（演示版数据包含真实的金额与多样的来源系统，如用友、SAP） 状态更新: 文件状态自动变为"已识别"
4. 上传成功后，文件会立即出现在列表顶部，显示唯一流水号（格式：POOL-YYYYMMDD-XXXXXXXX）
5. 正式归档 (新功能): - 在凭证池列表中，勾选确认无误的凭证。 点击工具栏上的 "正式归档" 按钮（绿色）。 系统将自动生成标准档号，封装 AIP 包，并将状态更新为"已归档"。 注：此操作适用于无需复杂关联的独立凭证快速归档。 核心归档流程图: graph TD A[电子凭证池] -->|1. 选择凭证| B(选中凭证) B -->|2. 点击按钮| C{正式归档} C -->|3. 系统自动处理| D[生成档号] D --> E[创建 AIP 包] E --> F[移动文件] F --> G[生成 metadata.xml] G --> H[更新状态] H -->|4. 完成| I((归档完成)) style C fill:#e1f5fe,stroke:#01579b,stroke-width:2px style I fill:#e8f5e9,stroke:#2e7d32,stroke-width:2px
步骤 2: 整理与关联 (Organize)
确保凭证与原始凭证（发票、回单）正确关联。
1. 进入 [预归档] -> [凭证关联]。
步骤 2: 整理与关联 (Organize)
确保凭证与原始凭证（发票、回单）正确关联，构建完整的证据链。
1. 进入 [预归档] -> [凭证关联]。
2. 智能匹配预演: - 点击 "自动匹配" 按钮。 系统弹出匹配预演窗口，展示"高置信度匹配"（绿色）和"疑似匹配"（黄色）结果。 确认无误后，点击 "确认并应用"，系统批量建立关联关系。
3. 关联全景视图: - 点击列表中的 "查看" (眼睛图标)，切换到 "关联全景" 页签。 您将看到以凭证为中心的可视化图谱：左侧为原始凭证（发票），右侧为依据文件（合同）。 深度穿透: 点击图谱中的任意卡片，或列表中的"关联发票数"，可直接查看单据详情。
4. 确认无误后，勾选已关联的条目，点击 "移交归档" 按钮（绿色）。
步骤 3: 正式归档 (Archive) 🆕 流程优化
从会计凭证页面直接完成归档。
1. 进入 **[档案管理]** → **[会计凭证]**
2. 在列表中找到 `draft` 状态的凭证（如 `YS-2025-08-记-1`）
3. **勾选** 要归档的凭证（可多选）
4. 点击右上角工具栏的 **"正式归档"** 按钮（绿色）
5. 系统将：
   - 执行四性检测
   - 锁定档案，禁止修改
   - 状态变更为 `archived`（已归档）
   - 生成归档日志
6. 归档完成后，可进行下一步：导出 AIP 包

步骤 4: 导出 AIP 包 (Export AIP)
生成符合 DA/T 94-2022 标准的归档信息包。

**方式一：单条凭证导出**
1. 在 **[档案管理]** → **[会计凭证]** 页面
2. 找到已归档 (`archived`) 的凭证
3. 点击操作列的 **下载图标 (📥)** 导出单条凭证的 AIP 包

**方式二：整卷导出**
1. 进入 **[档案管理]** → **[档案组卷]** 页面
2. 找到对应的案卷（如 `BR01-AC01-202508`）
3. 点击案卷卡片上的 **"导出 AIP"** 按钮
4. 下载 ZIP 格式的完整案卷 AIP 包

**AIP 包结构** (符合 GB/T 39674):
```
AIP_Root/
├── index.xml          # 总索引文件
├── /content           # 主件区 (OFD/PDF 版式文件)
├── /attachment        # 附件区 (发票、合同等)
├── /data              # 数据区 (accounting.xml)
└── /metadata          # 元数据区 (volume.xml)
```

步骤 5: 检索与利用 (Retrieve)
1. 进入 [档案查询] -> [全文检索]。
2. 输入关键词（如摘要、金额）查找档案。
3. 点击 "预览" 查看原文，或点击 "借阅" 发起下载申请。 > 💡 增强预览功能: 系统针对不同类型的档案提供专业化预览视图： 会计凭证: 展示发票详情、水印、电子印章。 会计账簿: 自动渲染为标准的三栏式或多栏式账页，清晰展示借贷余。 财务报告: 智能识别报表类型，自动切换 资产负债表 或 利润表 模板。

4. 功能模块详解
4.1 预归档模块 (Pre-Archive)
处理未归档的散乱文件。
- 电子凭证池: 文件的临时存放区。支持批量上传、删除、预览。 "上传" vs "新增" 的区别 这两个功能针对的业务场景完全不同：
📤 上传 (Upload): 场景: 您手头有现成的电子文件（如 OFD/PDF 发票、XML 凭证包）。 作用: 系统会自动读取文件内容，进行 OCR 识别、哈希计算 和 四性检测。这是批量处理的主要入口。 自动化程度: 高。系统自动提取元数据（如金额、日期）。 ➕ 新增 (New): 场景: 您需要归档一份实体纸质档案，或者文件尚未数字化，需要先占位。 作用: 您需要手动填写档案的元数据（如凭证号、摘要、金额），相当于手工录入一条目录信息。 自动化程度: 低。完全依赖人工录入。
- OCR 识别: 查看后台 OCR 任务进度。支持人工校对识别结果，修正错误的金额或发票号。
- 异常数据: 存放格式错误、哈希校验失败或无法识别的文件，需人工处理。
4.2 资料收集 (Collection)
对接外部系统。
- 在线接收: 集成监控台，实时监控外部系统的凭证同步状态。 支持系统: SAP ERP, 金蝶云星空, 用友 U8, 泛微 OA, 易快报, 汇联易等。 监控指标: 同步频率、最后同步时间、本次接收数量、接口健康状态。 操作: 支持手动触发同步、查看接口日志、新增集成接口配置。
- 扫描集成: 对接高拍仪/扫描仪，处理纸质凭证的数字化副本。
4.3 档案管理 (Archive Mgmt)
核心归档作业区域。
- 档案装盒: 以"卷"为单位管理。支持打印卷内目录、备考表。
- 会计凭证/账簿/报表: 分类查看不同类型的会计资料。
- 归档审批: - 待审批: 查看所有申请归档的案卷。管理员需检查四性检测结果。 批准/拒绝: 批准后档案正式归档；拒绝需填写理由，退回整改。
- 开放鉴定: - 鉴定任务: 系统自动筛选保管期限届满或需调整密级的档案。 鉴定操作: 判定档案是否可以向社会开放（完全开放/部分开放/不开放）。 延期保管: 对仍有价值的档案延长保管期限。
- 关联查看: 可视化展示"凭证-发票-回单"的挂接关系。
- 销毁鉴定: 对已到期的档案（如满10年）进行鉴定，生成销毁清册或延长保管期限。
4.4 全库检索与查询 (Global Search & Query)
- 顶部全局搜索 (DigiVoucher Search):
  - 入口: 页面顶部导航栏中央的搜索框，支持在任意页面随时唤起。
  - 快捷键: 
    - Mac 用户: Cmd + K
    - Windows 用户: Ctrl + K
  - 搜索能力: 支持模糊匹配 凭证号、摘要、金额 (精确匹配)、发票代码、往来单位 等。
  - 智能结果:
    - 档案匹配: 匹配到凭证或案卷标题 (如 "付款凭证")，图标显示为蓝色文档。
    - 元数据匹配: 匹配到具体的发票号或金额 (如 "12800.00")，图标显示为黄色代码，并高亮匹配详情。
- 档案查询模块:
  - 高级检索: 支持组合条件筛选（如：年度 + 会计科目 + 金额范围）。
  - 穿透联查: 从"总账"穿透到"明细账"，再穿透到"记账凭证"和"原始凭证"。
4.5 借阅利用 (Borrowing)
- 借阅申请: 普通用户填写借阅理由、期限，申请下载或打印权限。
- 我的借阅: 查看已获批的档案，支持在线水印预览和下载。
4.6 库房管理 (Warehouse)
- 实体库房: 管理纸质档案的存放位置（排/架/层）。
- 温湿度监控: 记录库房环境数据，确保实体档案安全。
4.7 档案全景 (Panorama)
- 档案目录树: 左侧提供按"年度-月份-凭证"层级的目录导航，快速定位目标档案。
- 多维关联视图: - 全景展示: 中间区域展示当前凭证的核心元数据。 证据链条: 右侧通过页签切换展示关联的"原始凭证"、"银行回单"、"合同文件"及"附件"。
- 真实感预览: - 合同预览: 支持查看包含条款、签字、盖章的高保真合同副本。 发票/回单: 还原真实的增值税发票和银行电子回单样式，支持查看明细与印章。
4.7.1 电子凭证翻阅器 (Voucher Player)
专为财务审核人员打造的沉浸式阅卷工具。
- 进入方式: 在"档案全景视图"右上角点击 "进入翻阅模式" 按钮。
- 核心功能: -
  1. 分屏对照: 左侧显示会计凭证，右侧显示关联的原始凭证（发票、合同），还原线下"左手翻凭证，右手对发票"的工作习惯。 
  2. 沉浸式翻页: 点击底部"上一张/下一张"或使用键盘 ←/→ 键，像翻书一样快速切换凭证。 
  3. X光透视 (X-Ray): 鼠标悬停在凭证的 金额 或 科目 上，右侧发票会自动 红框高亮 对应的数据区域，辅助快速核对。 
  4. 无级缩放: 滚动鼠标滚轮可对右侧附件进行无级缩放，按住鼠标左键可拖拽平移。 
  5. 安全水印: 界面覆盖"操作员+时间"的动态水印，顶部显示"四性检测通过"徽章，确保数据安全可信。
4.8 系统设置 (Settings)
- 用户管理: 添加用户、重置密码。
- 角色权限: 配置不同角色的菜单和操作权限。
- 审计日志: [强制] 所有关键操作（登录、归档、下载、修改）均会被记录，不可删除，满足审计合规要求。

5. 常见问题 (FAQ)
Q: 四性检测一直不通过怎么办？ A:
1. 真实性失败: 检查文件是否被篡改（哈希不匹配）。
2. 完整性失败: 检查是否缺少关键元数据（如凭证号）。
3. 可用性失败: 文件可能损坏，尝试重新上传。
Q: 如何修改已归档的档案？ A: 已归档档案严禁修改。如发现重大错误，需走"档案冲销"流程，并重新生成新的凭证进行归档。
Q: 为什么我看不到"销毁鉴定"菜单？ A: 该功能仅对"档案管理员"角色开放。请联系系统管理员核实您的权限。

## 6. 用友云凭证同步 (YonSuite Integration) 🆕

### 6.1 功能概述
系统支持从用友云 YonSuite 自动拉取会计凭证数据，实现 ERP 系统与电子档案系统的无缝对接。

### 6.2 同步操作步骤

#### 方式一：前端界面操作
1. 进入 **[资料收集]** → **[在线接收]**
2. 找到 **"用友YS凭证同步"** 接口
3. 点击 **"手动同步"** 按钮
4. 在弹窗中填写：
   - **账簿代码**: 例如 `BR01`
   - **起始期间**: 例如 `2025-08`
   - **结束期间**: 例如 `2025-09`
5. 点击 **"开始同步"**
6. 同步完成后，凭证自动进入预归档库

#### 方式二：API 调用
```bash
curl -X POST http://localhost:8080/api/integration/yonsuite/vouchers/sync \
  -H "Content-Type: application/json" \
  -d '{
    "accbookCode": "BR01",
    "periodStart": "2025-08",
    "periodEnd": "2025-09"
  }'
```

### 6.3 同步结果
- **档号生成**: 自动生成格式 `YS-[年月]-[凭证字]-[凭证号]`，例如 `YS-2025-08-记-1`
- **元数据映射**: 自动提取凭证金额、日期、摘要、制单人等信息
- **分录详情** (2025-12-05 新增): 完整同步凭证分录的 **摘要** 和 **科目** 信息
- **状态**: 同步后凭证状态为 `draft`（草稿），等待归档

### 6.4 验证同步数据 🆕

同步完成后，建议执行以下验证步骤确保数据准确：

#### 步骤 1：查看同步凭证
1. 进入 **[档案管理]** → **[会计凭证]**
2. 找到新同步的凭证（如 `YS-2025-08-记-1`）
3. 确认状态为 `draft`

#### 步骤 2：归档凭证
1. 勾选要归档的凭证
2. 点击 **"正式归档"** 按钮
3. 状态变为 `archived`

#### 步骤 3：导出 AIP 包验证
1. 点击已归档凭证的 **下载图标**
2. 解压下载的 AIP 包
3. 打开 `content/` 目录中的 PDF 文件
4. **核对以下信息是否与 YonSuite 一致**：
   - **摘要**: 如 "内部交易出库核算"、"销售出库核算"、"确认应收" 等
   - **科目**: 如 "6401 主营业务成本"、"112201 应收账款_贷款" 等

> ✅ **验证通过标准**: AIP PDF 中的摘要和科目应与 YonSuite 源系统完全一致

### 6.5 注意事项
> ⚠️ **重要提醒**:
> - 同步操作具有幂等性，重复同步同一凭证不会产生重复数据
> - 同步前请确认用友云账号权限和网络连通性
> - 建议在业务低峰期执行大批量同步

## 7. 档案组卷管理 (Volume Management) 🆕

### 7.1 功能概述
**组卷** 是将同一会计期间内的电子凭证按月归集成案卷的过程，是正式归档的必要步骤。
系统遵循 **DA/T 104-2024《ERP系统电子文件归档和电子档案管理规范》** 实现自动组卷。

### 7.2 进入组卷模块
1. 登录系统
2. 左侧菜单找到 **[档案管理]**
3. 点击展开子菜单
4. 点击 **[档案组卷]**

### 7.3 按月组卷操作

#### 步骤 1：发起组卷
1. 点击右上角 **"按月组卷"** 蓝色按钮
2. 在弹出的对话框中，选择 **会计期间**（如 `2025-08`）
3. 点击 **"开始组卷"**

#### 步骤 2：组卷结果
系统自动执行以下操作：
- ✅ 查找该期间所有未组卷的凭证
- ✅ 创建案卷记录
- ✅ 生成案卷号（格式：`[全宗号]-[分类号]-[期间]`，如 `BR01-AC01-202508`）
- ✅ 生成案卷标题（格式：`[责任者]+[年月]+会计凭证`，如 `泊冉演示集团2025年08月会计凭证`）
- ✅ 计算保管期限（取卷内最长）
- ✅ 关联凭证到案卷

#### 案卷号编码规则

| 字段 | 含义 | 示例 |
|------|------|------|
| 全宗号 | 组织/公司代码 | BR01 |
| 分类号 | 档案门类代码 | AC01 (会计凭证) |
| 期间 | 年月 (YYYYMM) | 202508 |

### 7.4 查看卷内文件
1. 在左侧案卷列表中，点击选中一个案卷
2. 右侧自动显示 **"卷内文件清单"**
3. 清单包含：序号、档号、题名、制单人、金额、日期、状态

### 7.5 归档审核流程

#### 步骤 1：提交审核
1. 组卷完成后，案卷状态为 **"草稿"**（灰色标签）
2. 确认卷内文件无误后，点击案卷卡片底部的 **"提交审核"** 黄色按钮
3. 案卷状态变为 **"待审核"**（黄色标签）

#### 步骤 2：审批归档
1. 档案管理员查看案卷信息
2. 确认无误后，点击 **"归档"** 绿色按钮
3. 系统自动执行：
   - ✅ 案卷状态变为 **"已归档"**（绿色标签）
   - ✅ 卷内所有凭证状态同步变为 **"已归档"**
   - ✅ 记录归档时间和审核人

#### 步骤 3（可选）：驳回
1. 如发现问题，点击 **"驳回"** 红色按钮
2. 填写驳回原因
3. 案卷状态回退为 **"草稿"**，可继续修改后重新提交

### 7.6 归档登记表
1. 案卷归档成功后，按钮变为 **"归档登记表"**（紫色按钮）
2. 点击查看符合 **GB/T 18894 附录 A** 格式的归档登记表
3. 登记表包含：
   - 登记号、案卷号、案卷标题
   - 全宗号、会计期间、分类
   - 保管期限、文件数量
   - 卷内文件清单（序号、档号、题名、日期、金额、制单人）

## 8. 常见问题补充 (FAQ Addendum)

**Q: 组卷失败，提示"该期间没有待组卷的凭证"怎么办？**

A: 这表示该期间的凭证已经全部组卷完成。请检查：
1. 是否选择了正确的期间
2. 该期间凭证是否已存在于其他案卷中
3. 可在案卷列表中查看已有案卷

**Q: 如何重新组卷？**

A: 需要先解除凭证与案卷的关联：
1. 删除现有案卷（仅限草稿状态）
2. 或联系系统管理员清理数据

**Q: 归档后发现凭证有误怎么办？**

A: 已归档案卷严禁修改。如确需更正：
1. 走"档案冲销"流程
2. 重新同步正确的凭证
3. 重新组卷归档
