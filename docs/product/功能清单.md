## 一、功能模块总览

| 模块编码 | 模块名称 | 核心职责 |
|:---|:---|:---|
| **PORTAL** | 档案门户 | 全局概览、待办提醒、快捷入口 |
| **PRE** | 预归档库 | 电子凭证采集、识别、关联与合规预检 |
| **COLL** | 资料收集 | 外部系统数据同步、手工上传与扫描集成 |
| **REPO** | 会计档案库 | 正式档案的四性检测、版本管理与长期保存 |
| **OPS** | 档案作业 | 组卷、装盒、归档审批与鉴定 |
| **UTIL** | 档案利用 | 全文检索、借阅流程、关联穿透与全景视图 |
| **DEST** | 鉴定销毁 | 档案到期鉴定、延期保管与销毁执行 |
| **STAT** | 统计分析 | 归档报表、利用率统计与合规性大屏 |
| **SYS** | 系统设置 | 组织架构、权限模型、安全策略与集成配置 |
