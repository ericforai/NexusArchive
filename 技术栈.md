# NexusArchive 技术栈文档

## 项目概述

NexusArchive 是一个现代化的电子会计档案管理系统，采用前后端分离架构，支持 AI 驱动的 OCR、智能关联和四性检测功能。

---

## 前端技术栈

### 核心框架
- **React** `19.2.0` - 用户界面构建库
- **TypeScript** `~5.8.2` - 类型安全的 JavaScript 超集
- **Vite** `6.2.0` - 下一代前端构建工具

### UI 框架与组件库
- **Tailwind CSS** (CDN) - 实用优先的 CSS 框架
  - 通过 CDN 引入 (`https://cdn.tailwindcss.com`)
  - 自定义主题配置（primary 色系、slate 扩展色）
  - Google Fonts: Inter 字体家族
- **Lucide React** `0.554.0` - 现代化图标库
- **Recharts** `3.4.1` - 数据可视化图表库
- **@xyflow/react** `12.9.3` - 流程图和关系图可视化

### 网络请求
- **Axios** `1.13.2` - HTTP 客户端

### 开发工具
- **@vitejs/plugin-react** `5.0.0` - Vite 的 React 插件
- **@types/node** `22.14.0` - Node.js 类型定义
- **ts-node** `10.9.2` - TypeScript 执行环境

### 构建配置
- **端口**: 5174 (开发服务器)
- **模块系统**: ESNext
- **目标环境**: ES2022
- **路径别名**: `@/*` 映射到项目根目录

---

## 后端技术栈

### 核心框架
- **Hono** `4.6.14` - 轻量级、高性能的 Web 框架
- **@hono/node-server** `1.13.7` - Hono 的 Node.js 服务器适配器
- **TypeScript** `~5.8.2` - 类型安全开发

### 数据库
- **Better SQLite3** `11.8.1` - 高性能的 SQLite 数据库
- **Drizzle ORM** `0.36.4` - 类型安全的 TypeScript ORM
- **Drizzle Kit** `0.28.1` - 数据库迁移和管理工具

### 认证与安全
- **@node-rs/argon2** `2.0.0` - 密码哈希算法（Argon2）
- **jose** `5.9.6` - JWT 令牌处理
- **nanoid** `5.0.9` - 唯一 ID 生成器

### 数据验证
- **Zod** `3.23.8` - TypeScript 优先的模式验证库

### 开发工具
- **tsx** `4.19.2` - TypeScript 执行和监听工具
- **@types/better-sqlite3** `7.6.12` - Better SQLite3 类型定义

### 服务器配置
- **端口**: 3001 (开发环境)
- **数据库路径**: `./data/nexusarchive.db`
- **模块系统**: ESModule
- **CORS**: 支持跨域请求（允许 localhost:5173, 5174）

## 开发工具链

### 包管理器
- **npm** - 依赖管理

### 脚本命令

#### 前端脚本
```bash
npm run dev      # 启动开发服务器 (端口 5174)
npm run build    # 构建生产版本
npm run preview  # 预览生产构建
npm run seed     # 运行数据库种子脚本
```

#### 后端脚本
```bash
npm run dev         # 启动开发服务器 (tsx watch)
npm run build       # TypeScript 编译
npm run start       # 启动生产服务器
npm run db:generate # 生成数据库迁移
npm run db:migrate  # 执行数据库迁移
npm run db:push     # 推送 schema 到数据库
npm run db:studio   # 打开 Drizzle Studio
```

### 前端环境变量

```env

CORS_ORIGIN=http://localhost:5174  # 允许的跨域源
JWT_SECRET=your_jwt_secret         # JWT 密钥
```

## 项目结构

```
nexusarchive/
├── frontend (根目录)
│   ├── components/          # React 组件
│   ├── utils/              # 工具函数
│   ├── hooks/              # 自定义 Hooks
│   ├── api/                # API 调用封装
│   ├── types.ts            # TypeScript 类型定义
│   ├── constants.tsx       # 常量配置
│   ├── App.tsx             # 主应用组件
│   ├── index.tsx           # 应用入口
│   ├── index.html          # HTML 模板
│   ├── vite.config.ts      # Vite 配置
│   └── tsconfig.json       # TypeScript 配置
│
└── backend/
    ├── src/
    │   ├── config/         # 配置文件（数据库等）
    │   ├── models/         # 数据模型和 Schema
    │   ├── controllers/    # 控制器
    │   ├── routes/         # 路由定义
    │   ├── middleware/     # 中间件
    │   ├── utils/          # 工具函数
    │   ├── scripts/        # 脚本（如数据种子）
    │   └── index.ts        # 服务器入口
    ├── drizzle/            # 数据库迁移文件
    ├── data/               # SQLite 数据库文件
    ├── drizzle.config.ts   # Drizzle 配置
    └── tsconfig.json       # TypeScript 配置
```
