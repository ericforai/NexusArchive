# 安全管理页面 (Security Pages)

本目录包含系统安全管理相关的页面组件，主要涉及跨全宗访问控制和授权管理。

## 目录结构

- `AuthTicketApplyPage.tsx`: 跨全宗访问授权票据申请页面。
- `AuthTicketListPage.tsx`: 授权票据列表和审批页面。

## 功能说明

### 跨全宗访问授权票据

#### 申请功能 (`AuthTicketApplyPage.tsx`)
- 选择目标全宗（自动过滤当前全宗）
- 设置访问范围：
  - 归档年度（可选）
  - 档案类型（可选）
  - 关键词（可选）
  - 访问类型（只读/读写）
- 设置有效期（1-90 天限制）
- 填写申请原因

#### 审批功能 (`AuthTicketListPage.tsx`)
- 查看授权票据列表（支持状态筛选）
- 查看授权票据详情（模态框展示）
- 第一审批和第二审批（两级审批流程）
- 撤销授权票据
- 状态管理和可视化（待审批、已批准、已拒绝、已撤销、已过期）

## 规范

1. 所有页面组件必须使用 TypeScript
2. 遵循 React Hooks 最佳实践
3. 使用 `lucide-react` 图标库
4. 统一的错误处理和加载状态展示
5. 模态框交互遵循无障碍设计原则
