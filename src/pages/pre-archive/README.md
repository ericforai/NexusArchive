# 预归档模块 (Pre-Archive Module)

**一旦我所属的文件夹有所变化，请更新我。**

预归档模块负责管理电子凭证池、单据池、OCR识别、凭证关联和异常数据等预归档功能，支持列表/看板双视图模式。

---

## 目录结构

| 文件 | 地位 | 功能 |
|------|------|------|
| `PoolPage.tsx` | 页面容器 | 电子凭证池容器页面，支持列表/看板双视图切换 |
| `PoolPage.css` | 样式文件 | PoolPage 组件的样式定义 |
| `PoolKanbanPage.tsx` | 页面组件 | 电子凭证池看板视图（通过 PoolPage 容器或旧路由访问） |
| `AbnormalDataView.tsx` | 页面组件 | 异常数据管理页面 |
| `OCRProcessingView.tsx` | 页面组件 | OCR 识别处理页面 |
| `manifest.config.ts` | 模块清单 | 模块导出清单与架构边界定义 |
| `__tests__/` | 测试目录 | 单元测试文件 |

---

## 功能说明

### 电子凭证池 (PoolPage)
- **列表视图**（默认）：适合查看大量数据，支持筛选、批量操作
- **看板视图**：直观展示处理流程（待检测→待补全→待归档）
- **切换方式**：点击页面右上角的"列表"/"看板"按钮，或使用 URL 参数 `?view=list|kanban`

### 其他功能
- **异常数据**：管理检测失败、数据异常的凭证
- **OCR识别**：智能识别发票、合同等单据信息
- **凭证关联**：将原始凭证与记账凭证进行关联
