# Product Definition: NexusArchive

## 1.0 Project Vision
NexusArchive 是一款专为企业级财务归档设计的私有化部署解决方案，严格遵循国家档案局 **DA/T 94-2022《电子会计档案管理规范》**。系统通过高度自动化的采集与预归档流水线，解决大型集团财务共享中心在海量数据下的合规管理难题。

## 2.0 Core Value Proposition
- **高性能合规**：支持大企业集团百万级记录的高并发四性检测与归档。
- **智能化自动化**：通过 AI 适配器与自动化流水线实现“低干预”甚至“零干预”的档案采集。
- **全链路安全**：深度集成国密算法（SM2/SM3/SM4）与哈希链防篡改技术。

## 3.0 Target Audience
- **大型企业集团财务共享中心**：处理海量、多源的财务档案。
- **合规与审计部门**：确保档案的真实性、完整性、可用性和安全性。

## 4.0 Key Features & Focus Area
- **自动化预归档流水线**：包括数据采集、自动分类、四性检测及凭证关联。
- **高性能检测引擎**：针对大数据量优化的异步检测机制。
- **智能化用户引导**：降低专业档案管理软件的学习成本，提供自动化操作建议。

## 5.0 Current Development Milestone

- **全链路验证与 Bug 修复**：核心功能采集流水线已跑通。当前重点为修复归档过程中的 SQL 类型冲突 Bug (VARCHAR vs BIGINT)，并实现物理归档动作的最终闭环。
